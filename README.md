<h1> Tweet Data </h1>	

<h2> Description </h2>

<p> This is a script that when run, will query the entire US and collect
	geo-data from tweeter users.</p>

<p> Type of data: </p>

<ul> 

<li> User ID </li>
<li> User Tweet Content </li>
<li> Geo LONG </li>
<li> Geo LAT </li>

</ul>
